# BiliVideoMerge

#### 介绍
哔哩哔哩视频合并工具

#### 软件架构

1. 读取json,xml文件获取视频信息
2. 遍历读取文件夹
3. 使用ffmpeg合并视频，考虑采用多线程
4. 弹幕转为字幕
5. 在线升级

#### 界面结构

上部三分之一区域

+ 选择目录
+ 输出目录
+ 合并按钮

下部

+ MVC视图，显示选择的目录列表文件，合并进度等信息
